# Essential Math for Data Science — Study Portfolio

This repository contains my personal study notes, exercise solutions and Python notebooks based on *Essential Math for Data Science* by Thomas Nield.

The goal of this project is to build a strong mathematical foundation for data science, machine learning and artificial intelligence, while also documenting my learning process in a clear and structured way.

![Book cover](images/book_cover.jpg)

## Why this project?

Mathematics is one of the core foundations behind data science and AI. This repository helps me strengthen the concepts that are often used in machine learning, statistics, optimization and data analysis.

Through this project, I practice:

- translating mathematical concepts into Python
- solving exercises step by step
- explaining formulas in plain language
- visualizing mathematical ideas
- building intuition for data science concepts
- improving my ability to communicate technical material clearly

This repository is part of my programming and data science portfolio.

## Topics covered

The notebooks cover topics such as:

- functions and variables
- linear algebra
- vectors and matrices
- probability
- statistics
- calculus
- derivatives
- integrals
- regression
- mathematical reasoning for machine learning
